What it is:

This program is a simulation of the ISO OSI network model.
It use differnt layers represented by classes.

How it works:

The program has the following classes to simulate the list:

-Application
-Presentation
-Transport
-Network
-Datalink
-Physical

Each class has its own responsebilities in the network.

Note:

Examples of use are provied in the main.cpp file.
Please enter network IPs and Ports correctly.
